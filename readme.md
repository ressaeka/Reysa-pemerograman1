# Program Pengeluaran Mingguan

## Deskripsi

Program ini digunakan untuk menghitung total pengeluaran selama satu minggu serta persentase tiap kategori pengeluaran (makan, transport, dan belanja).

## Cara Kerja

Pengguna menginput pengeluaran untuk hari Senin, Selasa, Rabu, dan Jumat.
Hari Kamis tidak ada pengeluaran (dianggap 0).

Program kemudian:

* Menjumlahkan seluruh pengeluaran
* Menghitung persentase tiap kategori

## Output

Menampilkan:

* Total pengeluaran
* Persentase makan, transport, dan belanja
